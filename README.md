# Landing Page COMETA

Landing page estática construida con HTML y CSS. Incluye el logo y la identidad visual de COMETA, junto con capturas del visor BIM del material fuente.


## Abrir

Abre `index.html` en cualquier navegador moderno.

## Archivos

- `index.html`: landing responsive.
- `assets/cometa-logo.png`: logo entregado por el usuario.
- `assets/*.png`: capturas de producto extraídas del material fuente.
